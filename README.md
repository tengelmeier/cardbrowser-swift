# CardBrowser - Swift 

Rudimentary macOS port of emvcardbrowser by Nic Bedford 
Implemented in Swift 5 for macOS 10.14

Debug help:

        sudo defaults write /Library/Preferences/com.apple.security.smartcard Logging -bool yes
